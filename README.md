# Ticket Classifier

A simple ticketing app that uses an LLM to classify tickets into categories.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Set your OpenAI API key:
```bash
export OPENAI_API_KEY='your-api-key-here'
```

3. Start the server:
```bash
npm start
```

4. Open http://localhost:3000 in your browser
